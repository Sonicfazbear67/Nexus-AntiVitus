# NEXUS Security - Advanced Threat Protection System

## Overview
Professional hacker-styled web antivirus software with comprehensive scanning capabilities. This is a frontend demonstration interface simulating real cybersecurity scanning operations.

## Features
1. **Link Scanner** - Analyze URLs for phishing, malware, and security threats
2. **File Scanner** - Upload and analyze files for potential malware
3. **Phone Number Scanner** - Check phone numbers for spam/fraud risks
4. **System Scanner** - Full system security diagnostic
5. **Virtual Test Environment** - Safe sandbox to test suspicious content
6. **Threat Analysis Dashboard** - Real-time security statistics

## Technologies Used
- HTML5
- CSS3 (with CSS Variables for theming)
- Vanilla JavaScript (no external dependencies)
- Font Awesome icons
- Google Fonts (Segoe UI, Consolas)
